Data

PostgreSQL

A complete PostgreSQL 9.5 dump (text format) of the Canadian Hansard dataset, including supplementary data. Updated on the last day of each month.

  • basehansard-1.0.2.sql.bz2

Click here for a basic tutorial on loading this dataset on Windows.

CSV (Comma-Separated Variable)

Download the full Canadian Hansard dataset in a series of daily UTF-8 CSV files.

  • lipadcsv-1.0.2.tar.bz2

NOTE Older versions of Excel may be unable to open these files because of French language characters.
We recommend using LibreOffice or OpenOffice to view CSV files.

XML

XML Canadian Hansard dataset digitized from original documents by the Lipad Group. 1901-1993 inclusive.

  • ca-proc.tar.gz

Supplementary Data

Additional data on parties and politicians in XML format.

  • ca-parties.tar.gz
  • ca-members.tar.gz

More Data Sources

  • openparliament.ca maintains a searchable record and PostgreSQL data dump of Canadian Hansard proceedings from 1994-.
  • canadiana.org hosts scanned, searchable images of the Debates and Journals of the Canadian House of Commons and Senate, from 1867 to 1994.
